Exercise 10.1 – Responsive Navbar

This navbar uses navbar-expand-lg — it shows the full menu on lg+ screens and collapses to a hamburger toggle on smaller screens.

ClassPurpose
navbarBase navbar styles
navbar-expand-lgCollapses on screens smaller than lg (992px)
navbar-darkWhite links/icons for dark backgrounds
navbar-brandLogo or site name styling
navbar-togglerThe hamburger button (hidden on large screens)
navbar-collapseWraps everything that collapses on mobile
nav-link activeHighlights the current page link
me-autoPushes nav items left; search goes to right
