Exercise 7.2 – Text Transform Utilities

Text Transform

text-uppercase — this text becomes all capitals

TEXT-LOWERCASE — THIS BECOMES ALL LOWERCASE

text-capitalize — every first letter gets capitalized

Text Alignment

text-start — Left aligned (default)

text-center — Center aligned

text-end — Right aligned

Responsive Text Alignment

text-center text-md-start — Centered on mobile, left-aligned on tablet+

text-start text-lg-end — Left on mobile, right-aligned on large screens

text-end text-sm-center text-md-start — Right → Center → Left

Line Height Utilities

lh-1 — Very tight line height for compact display text.

lh-sm — Slightly tighter than default.

lh-base — Default comfortable reading line height.

lh-lg — Generous spacing between lines for airy layout.
