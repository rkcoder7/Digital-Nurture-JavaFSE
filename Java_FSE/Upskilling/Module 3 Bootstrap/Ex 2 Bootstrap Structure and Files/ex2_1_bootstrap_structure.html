Bootstrap 5 – Directory Structure

📁 css/ Folder

Contains all the stylesheets. Key files:

FilePurpose
bootstrap.min.cssMain minified stylesheet — use in production
bootstrap-grid.min.cssGrid system only (lighter)
bootstrap-reboot.min.cssCSS reset/normalize only
bootstrap-utilities.min.cssUtility classes only
bootstrap.rtl.min.cssRight-to-left language support
📁 js/ Folder

Contains JavaScript for interactive components (modals, dropdowns, tooltips, etc.):

FilePurpose
bootstrap.bundle.min.js⭐ Recommended — JS + Popper.js bundled
bootstrap.min.jsJS only — needs separate Popper.js
bootstrap.esm.min.jsES Module format for modern bundlers
📁 Bootstrap Icons (Separate Package)

Bootstrap Icons is a separate library — not included in the Bootstrap download.

Install: npm install bootstrap-icons

CDN: https://cdn.jsdelivr.net/npm/bootstrap-icons/font/bootstrap-icons.css

Contains 2000+ SVG icons usable via <i class="bi bi-icon-name"></i>
