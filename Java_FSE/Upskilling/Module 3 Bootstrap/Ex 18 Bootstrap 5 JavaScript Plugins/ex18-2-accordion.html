You can include Bootstrap via CDN (add link/script tags to your HTML), via npm (npm install bootstrap), or by downloading the compiled files from the official Bootstrap website. For JavaScript functionality, include bootstrap.bundle.min.js which bundles Popper.js.

Bootstrap 5 provides extensive color utilities via contextual classes like bg-primary, text-success, etc. You can also fully customize the color palette using Sass variables.

Bootstrap 5's grid system is built entirely on CSS Flexbox. The utility classes like d-flex, justify-content-*, and align-items-* map directly to flexbox properties.
