Exercise 19.1 – Bootstrap 5 + Sass Project Setup

Follow the steps below to scaffold a Bootstrap 5 project that uses Sass for custom theming. After setup, you can override any Bootstrap variable before importing the framework.

1
Create project & initialise npm
mkdir my-bootstrap-sass-project
cd my-bootstrap-sass-project
npm init -y
2
Install Bootstrap & Sass
npm install bootstrap
npm install --save-dev sass

Bootstrap's npm package includes the full Sass source in node_modules/bootstrap/scss/.

3
Create scss/custom.scss
// scss/custom.scss
$primary:       #6f42c1;    // change primary from blue to purple
$secondary:     #20c997;    // teal secondary
$font-size-base: 1rem;
$border-radius:  0.5rem;    // more rounded corners


@import "../node_modules/bootstrap/scss/bootstrap";
4
Add npm scripts in package.json
{
  "scripts": {
    "build-css": "sass scss/custom.scss dist/css/custom.css --no-source-map",
    "watch-css": "sass scss/custom.scss dist/css/custom.css --watch"
  }
}
5
Compile the Sass
# One-time build:
npm run build-css
# Watch mode (recompiles on every file save):
npm run watch-css
6
Link the compiled CSS in your HTML
<link rel="stylesheet" href="dist/css/custom.css"/>
<!-- No more CDN needed — your compiled file IS Bootstrap + your theme -->
✅ What you get

After compilation, dist/css/custom.css contains the full Bootstrap CSS with your custom colors baked in. Every Bootstrap component — buttons, cards, badges, alerts — will automatically use your new $primary and other overridden variables.

Project Structure
my-bootstrap-sass-project/
├── node_modules/
│   └── bootstrap/
│       └── scss/          ← Bootstrap Sass source (read-only, don't edit)
├── scss/
│   └── custom.scss        ← YOUR entry file
├── dist/
│   └── css/
│       └── custom.css     ← compiled output (gitignore this)
├── index.html
└── package.json
